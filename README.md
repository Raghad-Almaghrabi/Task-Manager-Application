# Task Manager Application

A full-stack Task Manager application built with Node.js, Express, PostgreSQL, and React that allows you to create, read, update, and delete tasks.

## Tech Stack
- **Frontend:** React, Axios
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL

Refer to the README inside each folder for setup instructions.
